Glossary

While reading this site, you might have seen some words that you might not immediately know the meaning of. Worry not, for we have made a list of the most important terms and their definitions:

os
OS is short for operating system. An operating system is the software installed on the phone's storage that enable your apps to communicate with the hardware of your phone.
cpu
The CPU (Central Processing Unit) is the brain of your smartphone. It handles all the instructions it receives from hardware and software running on the phone.
ram
RAM (Random Access Memory), also referred to as system memory is a type of memory that is ultra-fast and is used to store the data of currently running applications.
