Responses

Web API requests and responses will always be in JSON format.

Response Status Code

Status Code Description Methods
200 Indicates that the request has succeeded GET, POST, PUT, DELETE
201 Indicates that the request has succeeded and a new resource
has been created as a result		 POST
204 The server has fulfilled the request but does not need to return a
response body. The server may return the updated meta information.		 PUT, DELETE
400 The request could not be understood by the server due to incorrect syntax.
The client SHOULD NOT repeat the request without modifications.		 GET, POST, PUT, DELETE
404 The server can not find the requested resource. GET, PUT, DELETE
500 The server encountered an unexpected condition that prevented it from
fulfilling the request.		 GET, POST, PUT, DELETE
